About selectors

These are the selectors that we will see today

  • Type Selectors: 
                        h1: selects all elements of type "h1"
  • Class Selectors 
                        .mashu: selects all elements with class=".... mashu ...""
  • Id Selectors 
                        #mashu: selects all elements with id="mashu"
  • Attribute Selectors 
                        [mashu=val]: selects all elements with attribute mashu with value val
                
                        [mashu]: selects all elements with attribute mashu
  • Wild card selector 
                        *: selects everything
